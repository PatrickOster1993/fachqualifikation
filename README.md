## 🚀 Installation von Python

1. Lade Python von der offiziellen Website herunter: [Python.org](https://www.python.org/downloads/)
2. Installiere Python und setze das Häkchen bei **"Add Python to PATH"**
3. Überprüfe die Installation mit dem Befehl:

   ```bash
   python --version
   ```

4. Falls du Pip benötigst, installiere es mit:

   ```bash
   python -m ensurepip --default-pip
   ```

---

## 🏁 Erste Schritte mit Python

Lass uns unser erstes Python-Programm schreiben! Erstelle eine Datei `hello.py` und füge folgenden Code hinzu:

```python
print("Hallo, Welt! 👋")
```

Speichere die Datei und führe sie mit folgendem Befehl aus:

```bash
python hello.py
```

Herzlichen Glückwunsch! 🎉 Du hast dein erstes Python-Programm geschrieben! 🐍
